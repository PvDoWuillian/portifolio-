Watering Kingdom

Bot completo de vendas de Robux • Primeiro cliente

✅ Entregue 🤖 Bot Discord 10 Comandos

Desenvolvi um bot completo e profissional para o servidor Watering Kingdom, contendo 10 comandos slash personalizados feitos sob medida para a equipe. O sistema inclui automação de vendas via PIX, painel administrativo, sistema de boas-vindas, avaliações e muito mais.

  • /ticket — Abre painel de compras (Admin)
  • /anuncio — Publica anúncios no servidor (Admin)
  • /tabela — Exibe tabela completa de preços de Robux
  • /planos — Mostra planos disponíveis para compra
  • /boasvindas — Envia mensagem de boas-vindas (Admin)
  • /avaliar — Sistema de feedback dos clientes
  • + 4 comandos exclusivos desenvolvidos para a equipe
